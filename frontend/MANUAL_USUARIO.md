# Manual de Usuario - EntryTech (Interfaz de Usuario)

Bienvenido al Manual de Usuario de **EntryTech**. Este documento detalla cómo utilizar las funciones de la interfaz para gestionar el control de acceso.

---

## 1. Inicio de Sesión (Login)
- **Acceso:** Ingrese a la plataforma utilizando su número de identificación (cédula) y su contraseña.
- **Seguridad:** El sistema utiliza tokens de sesión protegidos. Si olvida su contraseña, contacte al administrador del sistema.

## 2. Dashboard Principal
El Dashboard es su centro de control donde podrá visualizar:
- **Resumen de Actividad:** Cantidad de usuarios activos y accesos recientes.
- **Búsqueda Rápida:** Utilice la barra de búsqueda para localizar usuarios por nombre o cédula. El buscador es inteligente y reconoce nombres incluso sin tildes.
- **Últimos Registros:** Lista de las últimas entradas y salidas registradas en tiempo real.

## 3. Gestión de Usuarios (Solo Administradores)
Desde el módulo de "Usuarios", podrá realizar las siguientes acciones:
- **Registrar Usuario:** Complete el formulario con los datos personales y asigne un rol (Administrador o Usuario).
- **Editar:** Modifique información existente de cualquier perfil.
- **Eliminar:** Retire usuarios del sistema de manera permanente.

## 4. Mi Carnet Digital
Cada usuario tiene acceso a su carnet digital:
- **Identificación:** Muestra nombre, cargo y foto del usuario.
- **Código Dinámico (QR/NFC):** Este código es el que debe presentarse en los puntos de acceso (Kioscos) para registrar la entrada o salida.

## 5. Simulador de Kiosco
El sistema incluye un simulador para los puntos de control físico:
- **Validación:** El guardia o sistema automático escanea el carnet.
- **Registro Automático:** Si el usuario es válido, se registra la hora y tipo de movimiento (Entrada/Salida) en la base de datos de manera instantánea.

## 6. Cierre de Sesión Seguro
- Para salir del sistema, haga clic en el botón de "Cerrar Sesión".
- **Confirmación:** El sistema le pedirá una confirmación para evitar cierres accidentales.
